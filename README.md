# Loan Approval Mini Suite

A beginner-friendly web project built using HTML, CSS, JavaScript, and Chart.js.

This project combines 3 modules in one app:

1. Loan Approval Dashboard
2. Loan Eligibility Checker
3. Loan Statistics Explorer

## Features

### Dashboard
- View loan dataset in a table
- Search, filter, and sort applicants
- Summary cards
- Approval and income charts
- Credit score comparison chart

### Eligibility Checker
- Enter applicant details
- Get likely approval result
- View eligibility score and reasons

### Statistics Explorer
- Explore loan insights with charts
- Filter by approval status and city
- View applicant details in a popup
- Top 5 cities applicant chart

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
