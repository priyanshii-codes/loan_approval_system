DATA + DASHBOARD + ELIGIBILITY

Welcome to the Loan Approval Mini Suite

A web development project built using loan approval data. This mini suite combines a dashboard, a loan eligibility checker, and a statistics explorer in one modern dark-themed web app.

📊 Loan Approval Dashboard

View all applicants in a searchable table, filter by city and approval status, and analyze summary cards and charts.

Go to Dashboard →

✅ Eligibility Checker

Enter applicant details like income, credit score, loan amount, years employed, and points to check whether the loan is likely to be approved.

Go to Eligibility Checker →

📈 Statistics Explorer

Explore visual insights from the dataset with charts, filters, and applicant information in a clean interactive view.

Go to Statistics Explorer →

About This Project

This project is built with a minimal tech stack: HTML, CSS, JavaScript, and Chart.js. It demonstrates how loan approval data can be turned into a small interactive web application with multiple connected pages.
